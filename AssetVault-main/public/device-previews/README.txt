AssestFlow device preview images
================================

These files are the product photos you uploaded in chat (copied from Cursor workspace storage).

Each asset type uses a JPG in this folder (see src/lib/devicePreview.ts).

To replace any photo later:
1. Save the new photo with the exact filename below (overwrite the existing file).
2. Refresh the app — no code change needed.

Filenames:
  monitor.jpg, keyboard.jpg, mouse.jpg, ups.jpg, printer.jpg,
  qr-scanner.jpg, network-switch.jpg, camera.jpg, nvr.jpg,
  network-rack.jpg, laptop-kit.jpg, attendance-machine.jpg,
  external-hdd.jpg, access-point.jpg, firewall.jpg,
  network-controller.jpg, laptop.jpg, desktop.jpg

Recommended: square or 4:3, at least 600px wide, JPG or WebP (rename to .jpg).
